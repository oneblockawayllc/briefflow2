Usage Guide

Design Principles

  • Consistent spacing using defined tokens (xs, sm, md, lg, xl, 2xl)
  • Use semantic color names from the palette
  • Follow typography scale for hierarchy
  • Apply shadows and borders consistently
  • Ensure all interactive elements have hover/focus states

Component Library

  • Atoms: Buttons, Badges, Inputs, Typography
  • Molecules: Cards, Input Groups, Chip Selectors
  • Organisms: Progress Bars, Score Meters, Step Indicators
  • All components support light/dark mode via CSS variables
  • Accessible markup with proper ARIA labels
